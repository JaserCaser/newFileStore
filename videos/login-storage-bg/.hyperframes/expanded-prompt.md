# Cloud Archive Flow

Style: refined Apple-like glass on `#f5f5f7`, using `#0071e3` as the primary motion accent with quiet file-type color signals from the FileStore UI palette. Typeface follows the application stack: `SF Pro Display, SF Pro Text, PingFang SC, Microsoft YaHei, Segoe UI, sans-serif`.

Rhythm: `breathe-build-organize-loop`. A single 12-second ambient composition with no hard cuts and no text narration.

Global rules:
- 1920x1080 composition rendered as 3840x2160 via HyperFrames `landscape-4k`.
- Center 620px wide vertical zone remains calm for the login card.
- Every decorative layer moves slowly: drift, orbit, breathe, trace, align.
- All file cards use abstract glyphs only, no readable file names.
- The final second visually returns close to the opening layout so browser looping is gentle.

Beat 1, 0-3s, quiet intake:
- BG: soft radial blue glow, faint storage grid, slow grain wash.
- MG: translucent file cards float in from left/right edges.
- FG: a small cloud core appears behind the future login card zone at very low opacity.
- Motion: float, fade, drift, line traces begin.

Beat 2, 3-6s, secure sync:
- BG: blue glow breathes brighter.
- MG: connector paths draw toward the cloud core.
- FG: check-pulse rings and tiny sync ticks appear.
- Motion: draw, pulse, settle.

Beat 3, 6-9s, organized archive:
- BG: grid gains structure near lower left and upper right.
- MG: cards align into small stacks and ordered shelves at the frame edges.
- FG: folder tabs and file-type color chips snap into place.
- Motion: cascade, align, lift.

Beat 4, 9-12s, loop reset:
- BG: glows soften back to the starting intensity.
- MG: stacks loosen into drifting cards.
- FG: connector paths fade down while endpoints remain faint.
- Motion: unspool, fade, breathe.

Negative prompt:
- Avoid dark cyber dashboards, neon tunnels, loud particles, stock cloud icons, readable personal data, fast camera moves, dense elements behind the form, or saturated purple-blue gradients.
